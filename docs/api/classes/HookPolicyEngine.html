    Class HookPolicyEngine

    Policy engine for evaluating hook decisions based on tool categories, risk patterns, and configuration profiles. Supports strict, balanced, and permissive profiles with custom overrides.

    Example

    const engine = new HookPolicyEngine({ profile: "strict" });
    const decision = engine.evaluatePreToolUse(payload, context);
    if (decision.decision === "deny") {
      console.log("Tool blocked:", decision.reason);
    }
